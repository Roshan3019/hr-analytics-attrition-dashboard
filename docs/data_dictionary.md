
# Data Dictionary – HR Analytics Dataset

| Column Name | Description |
|------------|-------------|
| Attrition | Indicates whether the employee left the organization |
| Age | Employee age |
| Gender | Male or Female |
| Department | HR, R&D, or Sales |
| JobRole | Employee job title |
| EducationField | Academic background |
| JobSatisfaction | Satisfaction score (1–4) |
| WorkLifeBalance | Work-life balance rating (1–4) |
| MonthlyIncome | Employee monthly salary |
| TotalWorkingYears | Total years of professional experience |
| YearsAtCompany | Tenure at the organization |
| YearsInCurrentRole | Time spent in current role |
| YearsSinceLastPromotion | Time since last promotion |
| YearsWithCurrManager | Time with current manager |
